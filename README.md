# 🌸 Receitas do Dia a Dia

Site de receitas simples, rápidas e acessíveis para o cotidiano. O projeto foi desenvolvido com foco em uma interface amigável, design leve e navegação intuitiva.

## 📌 Sobre o projeto

O site é composto por duas páginas principais:

* **Página inicial:** apresenta uma interface de boas-vindas com um botão de acesso às receitas.
* **Página de receitas:** exibe receitas organizadas em formato de cards, contendo vídeo, descrição e lista de ingredientes.

As receitas disponíveis incluem:

* 🍝 Macarrão cremoso
* 🍫 Bolo de chocolate
* 🥟 Pastel simples

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

## 🎨 Funcionalidades

* Layout moderno e responsivo
* Cards organizados para cada receita
* Vídeos incorporados do YouTube
* Navegação entre páginas (início ↔ receitas)
* Efeitos visuais (hover, sombras e animações leves)

## 📱 Responsividade

O site foi desenvolvido para funcionar bem em:

* Computadores
* Tablets
* Celulares
